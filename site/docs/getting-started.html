Getting started

1. Start Postgres

$ cd docker && docker-compose up -d

2. Start the broker

$ export DATABASE_URL=postgres://postgres:postgres@localhost:5433/fluorite $ fluorite-broker

3. Create a topic and start streaming

# Bootstrap your API key $ fluorite bootstrap $ export FLUORITE_API_KEY=tb_... # Create a topic $ fluorite topic create my-events # Write events $ fluorite write --topic my-events '{"name": "alice", "amount": 100}' # Tail events in real-time $ fluorite tail --topic my-events
